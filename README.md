## Описание

GitHub User Finder — это десктопное приложение на Python с графическим интерфейсом (Tkinter), которое позволяет:

- Искать пользователей GitHub по логину через официальное REST API.
- Просматривать результаты поиска в удобном списке.
- Добавлять понравившихся пользователей в избранное.
- Сохранять избранное в локальный JSON-файл.
- Управлять избранными (удалять, просматривать).
